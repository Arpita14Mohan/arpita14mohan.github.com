Tips on CSS


While creating a website I found some interesting tips on CSS which I would like to share. These techniques were gathered after viewing CSS of different websites.

Use of li/ul tags to create Menu & styling through CSS -

This will create a left side navigation sidebar. The unordered list is styled using "list-style-type:none", this way the bullets won't appear for the list content.

<html>
<body>
<style type="text/css">
#left {
    float: left;
    width: 200px;
    padding: 0px 10px 40px 2px;
    line-height:150%;
}
</style>
<div id="left">
        <h2 style="color: Blue;">Blog Archive</h2>
        <ul style="list-style-type:none">
            <li><a href="#" >Design Learnings</a></li>
            <li><a href="#" >Vim Examples</a></li>
            <li><a href="#" >Version Control</a></li>
            <li><a href="#" >Unix Commands</a></li>
        </ul>
    </div>
</body>
</html>

Text Transform Property -

We often know them but forget to use them. Different text styling -
  • uppercase
  • lowercase
  • capitalize

<html>
<head>
<style type="text/css">
p.uppercase {text-transform:uppercase;}
p.lowercase {text-transform:lowercase;}
p.capitalize {text-transform:capitalize;}
</style>
</head>
<body>
<p class="uppercase">This is some text.</p>
<p class="lowercase">This is some text.</p>
<p class="capitalize">This is some text.</p>
</body>
</html>


Repetition of a image in a column

I have used the inserted image as a background for the complete web page.  This image is styled by using "repeat - y" property.It shows a complete demarcation between the left side and the right side.


<html>
<body>
<style>
body {
    background: #666666 url(images/back.png) repeat-y center top;
}
</style>
</body>
</html>
