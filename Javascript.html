Hello World - Javascript


Hello World,
This simple  blogs explains the simple steps involved in executing a javascript. Lets start with a small Hello World example.

Hello World!!!

<html>
<head>
<title>Hello World</title>
</head>
<body>
<script type="text/Javascript" >
    document.write('Hello World!');
</script>
</body>
</html>

When you run this code you should get just a simple “Hello World!” displayed. When the page is loaded the <script> tag tells the browser that there is “code” inside and the type attribute tells the browser that it is Javascript.

document.write('Hello World!');

document is a Javascript object, it contains all of the pages document information. print is a document method that prints the passed string onto the page.

The ; at the end notifies to the parser that it has reached the end of that line, and to move onto the next.

DOM (Document Object Model)
Anytime you add Javascript to your web project, it is usually to affect your DOM. For example: hiding/showing a div, changing some text in an element, changing a class of an element, creating a sliding image gallery, HTML elements being manipulated in some way.

To walk the DOM tree

    document
       | - body
       | - documentElement
       | - getElementById('element id')
       | - getElementsByTagName('element_name')
       
body ->The BODY element node (read-only)
documentElement -> The HTML element node (read-only)
getElementById('element id') -> Returns a reference to the element with the specified id
getElementsByTagName('element_name') -> Returns a collection of all elements with the specified element name
