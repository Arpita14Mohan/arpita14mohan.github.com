Dealing with Multiple Files

You can specify multiple files on the command line, as follows:

$ vim one.c two.c

This command starts Vim and tells it that you will be editing two files. By default,Vim displays just the first file. To edit the next file, you need to change files using the mentioned command.

:next

If you have unsaved changes in the current file and you try to do a :next, you will get a warning message and the :next will not work. You can solve this problem and save the file using the following command:

:write

In other words, you can perform a :write followed by a :next. The Vim editor has a shorthand command for this.The following command performs both operations:

:wnext
